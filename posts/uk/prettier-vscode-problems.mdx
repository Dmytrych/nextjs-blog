## Опис проблеми

Ви встановили та увімкнули Prettier у вашому JS застосунку, але Visual Studio Code не форматує та не підсвічує ваш код.

Якщо ж ви ще не встановили Prettier у ваш застосунок - подивитися гайд по його встановленню можна [за посиланням](/prettier-setup)

## Способи вирішення

Є кілька можливих проблем, які можуть трапитися із аналізом коду. Ось основні з них:



## Відсутність Підключеного Prettier

Перевірте ваш `package.json` на наявність встановленої версії Prettier:

<Image
    alt={`Prettier in package.json`}

У разі якщо prettier таки встановлений, спробуйте запустити команду `npm install`, про всяк випадок.



## Не Встановлене Розширення для VSCode

Переконайтеся що ви встановили розширення Prettier у VSCode. У разі якщо воно відсутнє - можете знайти його за посиланням: https://marketplace.visualstudio.com/items?itemName=esbenp.prettier-vscode



## Відсутність Файлу Конфігурації або Помилка в Ньому

Перевірте наявність і правильність файлу `.prettierrc` або `prettier.config.js`.

Спробуйте замінити свій файл конфігурації іншим, перевіреним, файлом. Можете використати базовий конфіг з документації:

```JSON
{
  "trailingComma": "es5",
  "tabWidth": 4,
  "semi": false,
  "singleQuote": true
}
```



## Prettier не Вказаний Стандартним Аналізатором Коду

Перейдіть у `File -> Preferences -> Settings` або (`Ctrl` + `comma`). І зробіть пошук по слову `formatter`.

У відкритій вкладці оберіть 'Default Formatter' - `Prettier`.

## Вимкнені Розширення Файлів або Неправильні Конфігурації Мови

Prettier може бути налаштований на вимкнення аналізу для певних типів файлів - тож перевірте, чи потрібні файли не включені у `prettier-ignore` файл.
Також, перевірте, чи правильно написане розширення вашого файлу.

Також переконайтеся, що налатування відповідають саме тій мові програмування яку ви використовуєте.



## Конфліктуючі Розширення VSCode

Спробуйте вимкнути інші розширення VSCode. Конфліктуючі розширення можуть заважати роботі Prettier.



## Застаріла Версія Prettier

Нові версії розширень VSCode можуть не підтримувати застарілі версії Prettier. В разі потреби, підніміть версію Prettier у `package.json`.
